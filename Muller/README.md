# Müller & Sohn Patternmaking Scripts

This folder contains Adobe Illustrator drafting scripts based on Müller & Sohn’s pattern construction system

## Structure

Müller/
├─ Bodice/
├─ Skirt/
├─ Sleeve/
└─ Trouser/

Each folder contains versioned Illustrator scripts (v1, v2, etc.) and a metadata.json describing the measurements and visual setup.

---

## Included Blocks

| Block   | Description                                             | Typical Use            |
| ------- | ------------------------------------------------------- | ---------------------- |
| Bodice  | Front and back foundations with darts and other shaping | Tops, dresses, jackets |
| Skirt   | Basic skirt foundations with darts                      | Skirts, dresses        |
| Sleeve  | Standard sleeve drafts                                  | Garments with sleeves  |
| Trouser | Trouser drafts with waist and hip shaping               | Trousers, shorts       |

---

## Requirements

- Adobe Illustrator

---

## Notes

- Units: centimetres
- Must have a basic understanding of Adobe Illustrator
- Some curves may come out badly. You must be ready to do some shaping
